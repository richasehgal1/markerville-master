ABOUT
the most comprehensive database of disease-biomarker pairings

Markerville is a web-hosted database that displays information about biomarkers and allows users to search and edit existing information, while adding new entries of their own. We are powered by Snorkel, an NLP pipeline package written in Python and developed by Stanford's Info Lab. We aim to create a community around biomarkers in the clinical research world and beyond. Thanks for visiting! More coming soon.
